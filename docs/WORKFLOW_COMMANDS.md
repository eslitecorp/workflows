# Workflow Commands

This document outlines the available workflow commands and their descriptions.

| Command | Description |
|---|---|
| `/invoke` | Runs the Gemini CLI |
| `/issue-fixer` | Fixes an issue with Gemini CLI |
| `/review` | Reviews a pull request with Gemini CLI |
| `/scheduled-triage` | Triages issues on a schedule with Gemini CLI |
| `/triage` | Triages an issue with Gemini CLI |
| `/plan` | Creates a plan, branch, and pull request without requiring approval. |
